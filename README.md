# Carbon Footprint Calculator & Tracker

A scalable, full-stack web application designed to help users track and reduce their carbon emissions through personalized AI recommendations.

## Features
- **User Authentication**: Secure Login/Register with JWT.
- **Activity Tracking**: Log Transportation, Energy, and Diet activities.
- **Visual Dashboard**: Real-time charts and stats (Recharts/Chart.js).
- **AI Recommendations**: Intelligent tips to reduce your footprint.
- **Premium UI**: Modern, responsive design with glassmorphism.

## Tech Stack
- **Frontend**: React (Vite), CSS Modules (Premium Design)
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Tools**: Lucide Icons, Chart.js, Axios
